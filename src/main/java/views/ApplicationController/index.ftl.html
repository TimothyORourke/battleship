The Battleship game

  • Place one of each ship to start the game.
  • Guess the position of enemy ships by clicking on top half of board.
  • To sink a ship, hit all squares that the ship occupies.
  • Game ends when one player sinks all the other players' ships.
  • Color codes:
  • ◼ MISS
  • ◼ HIT
  • ◼ SINK
  • ◼ PLACED
